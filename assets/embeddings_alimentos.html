01 — O que é um Embedding?

Um embedding transforma palavras, frases ou documentos inteiros em uma lista de números (vetor). Esses números capturam o significado semântico — palavras com significados próximos ficam próximas no espaço vetorial.

🥑
Texto original
"Abacate é rico em gordura monoinsaturada"
⟶
🔢
Embedding (simplificado)
[0.8, −0.2, 0.9, 0.1, −0.5, 0.7, ...]

02 — Por que Nutrição precisa de Embeddings?

Uma IA genérica não sabe distinguir uma recomendação clínica especializada de uma receita qualquer da internet. Os embeddings resolvem isso indexando o conhecimento específico da nutricionista.

🔍
Busca por Significado
A IA entende que "gordura boa para o coração" e "ácidos graxos ômega-3" são a mesma coisa, mesmo sem a palavra exata.
📋
Conhecimento Especializado
Os documentos da Dra. Patricia são indexados: protocolos, planos alimentares, contraindicações específicas.
🎯
Respostas Personalizadas
A IA recupera exatamente o protocolo certo para cada perfil de paciente (diabetes, emagrecimento, gestante...).
🛡️
Sem "Invenção"
Sem embeddings, a IA "alucina". Com eles, ela só responde com base nos documentos reais da nutricionista.
03 — Como a Similaridade Funciona

Quando o paciente pergunta algo, a pergunta também vira um embedding. O sistema compara matematicamente com os documentos indexados e recupera os mais próximos.

📩 Pergunta do paciente:

"Quais alimentos posso comer para reduzir o colesterol?"

→ Vira vetor: [0.7, 0.3, −0.2, 0.8, 0.5, ...]

04 — Pipeline Completo: RAG para Nutrição

RAG = Retrieval Augmented Generation — a IA primeiro busca nos documentos indexados, depois gera a resposta com base no que encontrou.

Fase 1 — Indexação (feita uma vez)

📁
Documentos
PDFs, protocolos, planos da Dra. Patricia
✂️
Chunking
Divide em trechos de ~500 tokens
🧮
Embeddings
Cada trecho vira um vetor numérico
🗄️
Vector DB
Vetores salvos (Chroma, Pinecone...)

Fase 2 — Consulta em tempo real

1
Paciente faz uma pergunta
"Tenho triglicérides alto, o que devo evitar comer?"
💬 Input
2
Pergunta vira Embedding
A pergunta é transformada em vetor com o mesmo modelo usado na indexação
🧮 Embedding
3
Busca Semântica no Vector DB
Sistema encontra os 3-5 trechos mais similares dos documentos da nutricionista
🔍 Retrieval
4
IA Gera a Resposta
Claude/GPT recebe: pergunta + trechos recuperados → gera resposta baseada no conhecimento da Dra. Patricia, não em dados genéricos da internet
✨ Generation
✅ Sua intuição está correta! Os embeddings são o "índice semântico" que permite à IA trabalhar como uma nutricionista experiente — não como uma IA genérica. Cada documento indexado é como um neurônio de memória especializado.
